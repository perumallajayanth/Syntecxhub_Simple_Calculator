# Syntecxhub Simple Calculator

## Project Overview

This project is a **Command-Line Simple Calculator** developed using **Python** as part of the **Syntecxhub Python Internship Program**.

The calculator performs basic arithmetic operations such as addition, subtraction, multiplication, and division. It accepts user input, processes the selected operation, and displays the result. The program also validates input and handles errors like division by zero.

---

## Features

* Addition (+)
* Subtraction (-)
* Multiplication (*)
* Division (/)
* Clear option
* Exit option
* Input validation
* Divide-by-zero error handling
* Menu-driven program for repeated calculations

---

## Technologies Used

* Python
* Command Line Interface (CLI)

## How to Run the Project

1. Install Python on your system.
2. Download or clone this repository.
3. Open the project folder.
4. Run the program using the following command:

```
python calculator.py
```
